# üè• Hospital Patient Records ‚Äì ML Analysis

Comprehensive machine-learning analysis of synthetic hospital patient records
generated by [Synthea](https://synthetichealth.github.io/synthea/).

## Dataset

| Table | Rows | Description |
|---|---|---|
| `patients.csv` | 974 | Demographics, birth/death dates, location |
| `encounters.csv` | 27 891 | Hospital visits with costs, payer, diagnosis |
| `procedures.csv` | 47 701 | Medical procedures linked to encounters |
| `payers.csv` | 10 | Insurance providers |
| `organizations.csv` | 1 | Massachusetts General Hospital |

## ML Tasks

| # | Task | Type |
|---|---|---|
| 1 | **Encounter-type classification** | Multi-class classification (6 classes) |
| 2 | **Mortality prediction** | Binary classification (imbalanced) |
| 3 | **Healthcare cost prediction** | Regression |
| 4 | **Patient segmentation** | Unsupervised clustering |

## Algorithms Used

- Logistic Regression, Decision Tree, Random Forest, Gradient Boosting, AdaBoost
- SVM (linear & RBF kernels), K-Nearest Neighbors, Na√Øve Bayes
- MLP Neural Network
- Voting & Stacking Ensemble classifiers
- Linear / Ridge / Lasso Regression, RF & GB Regressors
- K-Means, DBSCAN, PCA

## State-of-the-Art Techniques

- Stratified K-Fold Cross-Validation
- GridSearchCV & RandomizedSearchCV hyperparameter tuning
- Class-imbalance handling (`class_weight='balanced'`)
- Permutation-based feature importance
- ROC / AUC analysis, Confusion matrices, Learning curves
- Comprehensive HTML report with embedded visualisations
